# test-app
Servidor web de prueba para el taller de despliegue de aplicaciones en la nube

## Cómo ejecutar

- Instalar entorno de desarrollo y ejecución:
    ```bash
    ./scripts/install
    ```

- Iniciar servidor localmente:
    ```bash
    ./scripts/run_local
    ```

- Iniciar servidor públicamente:
    ```bash
    ./scripts/start
    ```
